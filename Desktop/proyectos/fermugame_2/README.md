# FermuGame

A retro RPG/adventure game set in a pharmacy with pixel art style, built with React, TypeScript, and Vitest using pnpm.

## Project Structure

- `src/` - Source code
  - `components/` - React components
  - `games/` - Minigame implementations
  - `utils/` - Utility functions
  - `test/` - Test setup
- `public/` - Static assets

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm dev
```

Run tests:

```bash
pnpm test
```

Build for production:

```bash
pnpm build
```
